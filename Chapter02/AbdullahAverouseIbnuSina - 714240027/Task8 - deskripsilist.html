Ppular Technology Terms

CPU
Central Processing Unit, or processor, is the main controller of the computer and is responsible for running the computer's processes.
Ram
Random Access Memory is a volatile memory that stores data and programs that the CPU is currently using. RAM is also known as the computer's short-term memory or working memory. Data stored in RAM is temporary and will be lost if the computer loses power or is restarted.
Rom
Read Only Memory is a non-volatile memory that stores data permanently, even when the computer loses power. ROM is used to boot up the computer and to store information about the computer's activities while it's running. ROM is also known as firmware or BIOS
